# todo

PWA 기반 투두 앱입니다.

## 소개

- React + TypeScript + Vite 기반
- 설치 가능한 PWA
- 오프라인에서도 기본 진입 가능
- 할 일 추가, 목록 확인, 완료 체크, 상세 편집 지원
- 커스텀 날짜 선택 달력 포함

## 현재 구현 범위

- 할 일 추가
- 할 일 목록
- 오늘 완료한 일 표시
- 상세 보기 및 메모/단계 편집
- 설정 화면
- 커스텀 달력: 연도 선택, 월 선택, 날짜 선택
- PWA `manifest` / `service worker`

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속.

## 테스트

```bash
npm test
```

## 빌드

```bash
npm run build
```
